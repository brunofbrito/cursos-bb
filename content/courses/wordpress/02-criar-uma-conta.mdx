# Como criar uma conta no WordPress.com

Para criares uma conta nesta plataforma, bastará registares-te com um *login* e uma *password*. 

Se já comentaste em sites WordPress, é possível que já tenhas uma conta criada (mesmo que nunca tenhas criado 1 site).
