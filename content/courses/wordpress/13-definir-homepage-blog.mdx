# Como definir Homepage e Blog no WordPress

Muitos sites não querem que a sua homepage seja uma lista dos artigos mais recentes. Quando assim é, precisamos de definir uma página como página principal.

Também é possível definir uma outra página como a página do Blog, ou seja, aquela que lista os artigos mais recentes (ordenados por data de publicação).
